# Real-Time Background Remover (Python)

A real-time background remover built using **Python, OpenCV, and CVZone Image Segmentation**.
This project captures video from the webcam and removes the background using AI-based human segmentation.

It demonstrates how **computer vision** can process live video and separate the foreground (person) from the background in real time.

---

## Features

* Real-time webcam background removal
* AI-based human segmentation
* Lightweight and fast processing
* Beginner-friendly Python project

---

## Technologies Used

* Python
* OpenCV
* CVZone
* MediaPipe Selfie Segmentation
* NumPy

---

## Installation

Clone the repository

```
git clone https://github.com/Ashadikram/realtime-background-remover.git
```

Navigate to the project folder

```
cd realtime-background-remover
```

Install dependencies

```
pip install opencv-python cvzone numpy
```
